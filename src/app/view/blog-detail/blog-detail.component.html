Multi-Level Category Selection options (NEW) :
Recently we have added Multilevel category selection option which determines the look of the categories field inside the submission form. This option will allow the user to select the category as per parent and child hierarchy. It is a very useful option when you have a number of categories and sub-categories to choose. Simply select a parent category it will open a new section which loads all child categories of that parent category. With this option, you can select multiple parent/child categories while submitting a listing. Isn’t it is more manageable?

Available choices are Checkbox, Multi-select, Select and Multi-Level Category Selection. Checkbox, Multi-select and Multi-Level Category Selection allow visitors to select multiple categories while Select allows only one to be selected.
